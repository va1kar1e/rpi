# Raspberry PI
About Raspberry PI

## Default Credential

```plaintext
Username: pi
Password: raspberry
```

## SSH 

```bash
ssh pi@raspberrypi.local
```

Once you have more than one Pi on your network, a number will be attached to the hostname.

```bash
ssh pi@raspberrypi-NUM.local
```
